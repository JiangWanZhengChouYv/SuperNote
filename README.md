# SuperNote

一个好用的在线记事本应用

## 版本
pre-1.0

### 版本说明
- **pre-1.0**: 初始版本
  - 实现基本的文本编辑功能
  - 支持文本格式化（粗体、斜体、下划线）
  - 本地存储和自动保存
  - 响应式设计
  - 字符计数功能
  - 通知系统

## 功能特点
- 文本编辑和格式化
- 本地存储，数据安全
- 自动保存功能
- 字符计数
- 响应式设计

## 如何使用
1. 在浏览器中打开 `index.html` 文件
2. 开始输入你的笔记
3. 使用工具栏进行文本格式化
4. 笔记会自动保存到本地存储

## 技术栈
- HTML5
- Tailwind CSS v3
- Font Awesome
- JavaScript
