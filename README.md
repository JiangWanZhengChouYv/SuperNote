## 如何使用
1. 在浏览器中打开 `index.html` 文件
2. **创建笔记**:
   - 点击左侧底部的"新建笔记"按钮
   - 选择"普通笔记"或"记账笔记"
3. **编辑笔记**:
   - 普通笔记: 在编辑器中输入标题和内容
   - 记账笔记: 填写金额、选择分类、设置日期和添加描述
4. **保存笔记**: 点击"保存"按钮或等待自动保存（每30秒）
5. **切换笔记**: 点击左侧列表中的笔记项
6. **删除笔记**: 鼠标悬停在笔记项上，点击出现的删除按钮
7. **导入导出**: 点击工具栏中的"设置"按钮，在设置界面中进行导入导出操作
